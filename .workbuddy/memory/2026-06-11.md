### 各优化器核心思路（备忘）
- **APE**：一次生成多候选 → 评估 → 返回最优
- **OPRO**：LLM 自我迭代，每轮用历史得分引导生成新 prompt
- **DSPy**：Bootstrap 采正确样本做 few-shot → LLM 提议新指令 → 评估
- **PromptBreeder**：进化算法，变异(mutation) + 交叉(crossover) 繁殖 prompt 种群
- **EVOPrompt**：结构化变异算子 + 早停，精英比例/变异率可配
- **SPO**：语义邻域搜索，多语义角度改写 prompt，引导式迭代优化
